# Tax Graph
This program will display a stacked line chart of net pay and tax amount for a range of biweekly wage amounts.

---

## How to Run
### In a terminal

1. Go to the project root directory, and install the required library.
```
   pip install matplotlib
```

2. Run the program.
```
   python tax_graph.py
```
### In Google Colab

1. Run this in a code cell.
```
   !pip install matplotlib
```

2. Paste the code in main.py into a code cell.  After the previous code cell finishes, run this code cell.
```
   python tax_graph.py
```
